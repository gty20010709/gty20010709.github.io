用Python绘制一只奔跑的小马

〇、 缘起

今天早上在吃早饭时，我刷了个视频，讲 Python 的，用一行代码打印一个复杂的三角形；代码是很好，但可读性一点是没有了。

在视频的末尾，作者在希望大家关注时，在控制台展现了一只小马，正在跑动，后面写着 “LIKE”、“SHARE”、“SUBSCRIBE”，我认为很有新意，反而是这个视频中最令我注目的地方。

不过作者没有放出代码，只是在视频中一闪而过；于是，我照着视频中的画面，自己绘制(抄制)了一只几乎完全的相同的小马；并实现了奔跑的效果。

此是为缘起。
