自建ip代理池之设想

〇、 缘起

在使用爬虫进行自动化仼务时，一个好用的 IP 池往往可以简化整个流程并促进目标的达成。

我现在在用的商用 IP 池，比较贵（20r/g流量）。因此，我想抽空研究一下自建免费 IP 池。

一、思路

REF: What is Public Proxy Pool & How to Build a Free Proxy Pool by Yourself?

summarized by Windows Copilot

  • 寻找免费代理网站：你需要找到一些提供免费代理列表的网站，例如 Proxyrack, Free Proxy List, name, Free Proxy Lists 等。你要确保选择的代理是高度匿名的，不会泄露你的真实 IP 地址。你找到的网站越多，你的代理池就越大。

  • 编写网络爬虫：你需要用编程语言编写网络爬虫，从免费代理网站上抓取代理的 IP 和端口，然后测试它们是否可用，再将它们存入数据库中。

  • 定期检查代理：你需要创建一个定时函数，每隔一段时间就检查数据库中的代理是否仍然可用，你可以用一个 API 如 http://httpbin.org/ip 来测试代理的状态。对于不可用的代理，你需要从数据库中删除它们。这一步很重要，因为免费代理通常会在一段时间后失效，因为 IP 的所有者可能会停用或网站会封锁它们。

  • 创建代理 API：你需要创建一个代理 API 来从数据库中获取可用的代理，你可以让这个 API 返回随机的代理，或者支持 IP 旋转，这样可以避免系统一直返回同一个代理而导致被封锁。
