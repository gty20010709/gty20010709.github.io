我的Windows和Linux纠结

〇、缘起

我的心又开始痒了，重装系统的念头又燃烧起来了。我得记录一下我为什么想要重装系统。

一、回到Windows（2023.7.27）

上次重装系统是从 Linux 回到 Windows，发生在7月底。

当时我用的是 KDE Neno， 基于Ubuntu的衍生版本。之所以离开这个版本，是因为：

  • Linux上的许多软件比较陈旧，我想要使用新的软件比较麻烦。
  • 国内高频使用的办公软件——微信，在Linux几乎不能使用。

日后想到了再补充。

二、回到Linux

现在（2023.9.1）， 我用的是 Windows 11， 我觉得在使用过程中有一些痛点让我想离开它。

  • 糟糕的终端体验。在Linux上，我习惯使用fish，有命令提示、命令高亮。

我好想找不出更多了，日后想到再补充。

三、结果

我先在备用机上重装系统，缓解一下心中的骚动，主力军要办公，还是得Windows。
