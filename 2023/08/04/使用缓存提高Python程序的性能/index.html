使用缓存提高Python程序的性能

〇、 缘起

昨天探索百度的云平台，许多服务在调用时都需要获取一个 access_token ，然后以之为凭证去访问 API。

百度的这个 access_token 有一个特点，就是存在一个月（30天）的有效期。

如果不使用缓存，在服务中每次调用 API 都获取的话，会造成极大的额外开销。 如果只在程序开头获取一次，而程序进行不间断运行的话，便无法更新 access_token (或许是我没有找到方法)。

此是为写这篇博文的缘起。
