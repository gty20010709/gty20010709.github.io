Crossref:开放的论文元数据检索API

零、 缘起

本需要研究从 Google Scholar 和 百度学术 抓取数据，但无奈人机验证和条目摘要难以提取，故寻找其他途径。在浏览 Sci-Hub 时， 从其 [database页面](https://sci-hub.se/database) 发现了 Crossref, 它提供开放的 API。

此是为缘起。

一、从 Corssref 检索文献

Crossref 是一个非盈利组织，提供文献元数据的检索服务，并提供免费的 API 服务，不需要登录便可以获得 50 次/s 的 API 调用权限。

Crossref 的 API 文档位于： [https://api.crossref.org/swagger-ui/index.html#/Works/get_works。

通过 Crossref 的 API， 可以凭关键词进行搜索，取得指定数目的文献信息，每条信息中，包含论文的标题、作者、URL、DOI、摘要（部分有）等信息。元数据示例

Crossref 的所有数据都可以从这里下载： https://academictorrents.com/checkb.htm?search=Crossref

Crossref 的不足

  • 中文文献少

二、Google Scholar 与 百度学术

在二者的网页进行搜索后，可以得到一个论文列表，在列表的每一行中，是论文的标题、作者、不完整的摘要等信息。

在 Google Scholar 上点击论文标题，会离开 Google 的网站，跳转到论文提供者的页面或者是直接下载 PDF，要获取完整的论文摘要，是十分困难的。

在百度学术上点击论文的标题，会进入一个详情页面，页面包含该论文的摘要。但使用程序在百度学术上搜索时，会频繁触发人机验证，暂未能解决；Google Scholar 也会触发人机验证，未能解决。
